yum -y install make zlib zlib-devel gcc-c++ libtool openssl openssl-devel

  • cd pcre-8.45/
  • ./configure
  • make && make install
  • 查看版本:pcre-configure -version

3、安装nginx

5、负载均衡配置 分配服务器的策略

  • 轮询，默认，每隔请求按时间顺序逐一分配到不同的后端服务器，如果后端服务器down掉了，能自动剔除
  • weight 代表权重，默认值是1，权重越高被分配的客户端越多

ip_hash 每隔请求按访问ip的hash结果分配，这样每个访客固定访问一个后端服务器

 upstream appstream {
     ip_hash;
     server localhost:8080 ;
     server localhost:8081 ;
 }
  • 按后端服务器的响应时间来分配请求，响应时间短的优先分配

再在server中配置proxy_pass

8、nginx 机制

  • 一个master 多个woker 的好处
    • 可以利用nginx -s reload 做热部署
    • 多个worker是独立的进程，其中一个worker有异常，不会影响其他的worker
    • worker = cpu数量 是最为合适
    • 每个worker的最大连接数是1024
  • 最大并发
    • 静态资源 worker_connections*worker_process/2 worker_process 只有接受和发送
    • 反向代理:worker_connections*worker_process/4 worker_process 需要反向代理 2次
